
# MaterialFlow Pro - 物料管理系统 (GitHub 最新代码同步版)

MaterialFlow Pro 是一款为中小型车间和门店设计的专业物料管理系统。系统专注于库存流转的实时计算、数据级联更新、审计日志以及定制化报表导出。

## 核心系统架构

1.  **数据持久化层 (SQLite 抽象)**：
    *   系统采用 Service 模式封装了数据库操作。在当前浏览器环境下，通过 `localStorage` 模拟高性能的 KV 数据库，并实现了复杂的**库存级联计算 (Cascading Updates)**。
    *   逻辑：修改历史某一天的库存数据，系统会自动重算该日期之后所有相关的期初与期末库存，确保账目永远平齐。
2.  **看板管理**：实时监控物料动态，支持搜索过滤与低库存自动预警。
3.  **批量化操作**：
    *   支持物料批量删除及相关历史数据一键清理。
    *   支持通过 Excel 批量导入物料库。
4.  **定制化导出**：导出 Excel 时可自由筛选字段（如：仅导出名称和剩余库存）。
5.  **安全管理**：
    *   基于环境变量的账号密码授权。
    *   详细的操作行为日志记录（审计追踪）。
